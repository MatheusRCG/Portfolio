Sobre Mim

Olá! Sou Matheus Rodrigues, engenheiro de software formado pela UFMS, apaixonado por tecnologia e inovação. Já atuei no desenvolvimento back-end, construindo APIs RESTful com Java e Spring Boot, e também como engenheiro de dados, trabalhando com Python, Spark e computação em nuvem na AWS. Mais recentemente, tenho aprofundado meus conhecimentos em desenvolvimento web, criando interfaces modernas e bem estruturadas.

No meio acadêmico, participei de projetos como o Ouvidor Mirim de Educação Ambiental, uma plataforma em React e TypeScript desenvolvida para a AGEMS, auxiliando na gestão de relatos feitos por crianças sobre problemas de saneamento. Também contribuí no Online Evidence Grabber, um software forense em Python criado para a Polícia Federal.

  • SELECIONE UMA HABILIDADE

  • HTML

    HTML (HyperText Markup Language) é a linguagem de marcação utilizada para estruturar e organizar conteúdo na web. Ele define elementos como títulos, parágrafos, links, imagens e tabelas em um site.

  • CSS

    CSS (Cascading Style Sheets) é a linguagem utilizada para estilizar elementos HTML, permitindo alterações em cores, espaçamentos, fontes e layouts responsivos.

  • JavaScript

    JavaScript é uma linguagem de programação que adiciona interatividade às páginas web, permitindo eventos dinâmicos, manipulação de elementos DOM e comunicação com servidores.

  • TypeScript

    TypeScript é um superset do JavaScript que adiciona tipagem estática ao código, ajudando na detecção de erros e na manutenção de grandes projetos.

  • React.js

    React.js é uma biblioteca JavaScript para criação de interfaces de usuário interativas, baseada em componentes reutilizáveis e gerenciada pelo Facebook.

  • Material UI

    Material UI é uma biblioteca de componentes React baseada no design system do Google, oferecendo componentes modernos e estilizados para aplicações web.

  • Node.js

    Node.js é um ambiente de execução JavaScript no servidor, permitindo a criação de aplicações escaláveis e performáticas com uso de eventos assíncronos.

  • Python

    Python é uma linguagem de programação versátil e fácil de aprender, amplamente utilizada em análise de dados, inteligência artificial, desenvolvimento web e automação.

  • Java

    Java é uma linguagem de programação orientada a objetos conhecida por sua portabilidade e segurança, sendo amplamente utilizada em aplicações corporativas e dispositivos Android.

  • Spring Boot

    Spring Boot é um framework Java para desenvolvimento rápido de aplicações web e microserviços, oferecendo configuração automática e segurança integrada.

  • MySQL

    MySQL é um sistema de gerenciamento de banco de dados relacional (RDBMS) de código aberto, amplamente utilizado para armazenar e manipular dados em aplicações web.

  • PostgreSQL

    PostgreSQL é um banco de dados relacional de alto desempenho e de código aberto, conhecido por sua conformidade com padrões SQL e suporte a extensões avançadas.

  • AWS

    AWS (Amazon Web Services) é uma plataforma de computação em nuvem que oferece serviços escaláveis para armazenamento, processamento e redes.

  • Docker

    Docker é uma plataforma para criação e gerenciamento de containers, permitindo a implantação rápida e consistente de aplicações em qualquer ambiente.

  • Git

    Git é um sistema de controle de versão distribuído que permite o rastreamento de alterações em código, facilitando o trabalho colaborativo entre desenvolvedores.

  • GitHub

    GitHub é uma plataforma baseada em Git para hospedar repositórios de código, facilitando a colaboração e o versionamento de projetos.

  • Swagger

    Swagger é uma ferramenta para documentação de APIs, permitindo a geração automática de interfaces interativas para testes e integração.
